# 서버가 꺼졌을 때 해야할 행위
1. Seetrol을 이용하여 고객사 PC에 접속합니다.(`[H&S]GatheringPC`)
2. 파일 탐색기를 실행합니다.
3. nginx 프로그램이 설치된 경로로(`C:\Program Files\nginx\1.22.0`) 이동합니다.
4. 파일 탐색기에서 nginx.exe 프로그램을 마우스로 더블클릭해서 실행합니다.
5. 웹 브라우저에서 화면이 나타나는지 확인합니다.
6. 서버가 실행되지 않을 경우 서비스를 재시작합니다.
